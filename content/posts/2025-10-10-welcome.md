# Welcome!

This is an example blog post written in **Markdown** and generated using a custom static site generator written in Go.

## Features

This SSG supports:

- GitHub Flavored Markdown
- Footnotes[^1]
- YAML frontmatter
- Draft posts
- Tags and metadata
- Smart typography
